# 错题复习工作台单页设计 Brief

## 页面定位

把当前“错题库”从工具首页里拆出来，设计成一个独立的学习工作台页面。页面核心不是展示一堆错题记录，而是帮助学生、家长和老师快速完成三件事：看见薄弱点、安排复习、生成针对性练习。

一句话定位：

```text
错题不是归档，是下一次练习的素材。
```

## 目标用户

- 学生：少整理、多复习，知道今天该练哪几题。
- 家长：不懂具体题目，也能看懂孩子最近错在哪里。
- 教培老师：按学生、知识点、错因快速筛选错题并生成练习。

## 核心页面叙事

页面应该从“今日该做什么”开始，而不是从“所有错题列表”开始。

推荐叙事顺序：

1. 今日复习概览：待复习、逾期、已掌握、本周新增。
2. 薄弱点诊断：知识点排行、错因分布、重复出错提醒。
3. 错题处理区：筛选、搜索、选择、查看详情。
4. 练习生成区：基于选中错题或薄弱知识点生成巩固卷。
5. 复习记录区：做对、又错、已掌握，形成下次复习时间。

## 页面模块

### 1. 顶部状态区

用途：让用户一眼知道当前孩子和今天要处理的学习任务。

建议内容：

- 当前孩子 / 年级 / 学科切换
- 今日待复习数量
- 逾期未复习数量
- 本周新增错题数量
- 主要按钮：拍试卷入库、生成今日练习

描述词：

```text
清晰的学习仪表盘、今日任务优先、低焦虑、行动导向、家长也能看懂
```

### 2. 薄弱点诊断区

用途：把错题从“题目列表”升级为“学习问题地图”。

建议内容：

- 薄弱知识点 Top 5
- 错因分布：审题错误、计算错误、概念不清、公式不会、表达不规范
- 重复出错提醒：同一知识点近期反复错误
- 可点击知识点直接筛选错题

描述词：

```text
知识点地图、错因洞察、弱项雷达、重复错误提醒、从数据走向行动
```

### 3. 错题列表区

用途：提供高效率检索和批量处理。

建议内容：

- 搜索题干、解析、标签
- 筛选：学科、年级、知识点、错因、题型、难度、状态
- 列表字段：题干摘要、知识点、错因、下次复习时间、掌握状态
- 批量操作：加入练习、标记已掌握、归档

描述词：

```text
可扫描、可筛选、可批量处理、信息密度适中、不要像后台表格一样冰冷
```

### 4. 错题详情区

用途：在同页内完成“看题、看错因、改标签、记录复习”。

建议内容：

- 原题题干和选项
- 学生答案、正确答案
- AI 解析
- 知识点、错因、题型、难度、标签编辑
- 复习记录时间线
- 操作按钮：做对了、又错了、已掌握、生成同类题

描述词：

```text
讲题卡片、复习记录、掌握度追踪、结构化标签、同类题入口
```

### 5. 练习生成区

用途：把错题转化成下一次练习，而不是只停留在收藏。

建议内容：

- 选择范围：今日待复习、选中错题、指定知识点、重复错误
- 题量：5 / 10 / 15 / 20
- 难度：简单 / 中等 / 困难 / 混合
- 题型：同类变式题、知识点专项题、易错辨析题
- 输出：在线预览、PDF 下载、打印

描述词：

```text
一键组卷、针对性练习、考前复习、同类变式、薄弱点专项
```

## 视觉方向

推荐风格：

```text
清爽可信、学习工具感、轻量仪表盘、信息有层次、温和但不幼稚、适合家长和中学生
```

避免方向：

```text
过度科技风、纯后台管理系统、颜色太童趣、营销落地页、巨型宣传 hero、满屏卡片堆叠
```

色彩关键词：

```text
白色留白、浅青绿色行动色、温和蓝色信息色、琥珀色提醒、少量红色用于逾期或高风险错因
```

布局关键词：

```text
左侧筛选列表、右侧详情抽屉、顶部任务概览、下方练习生成、桌面三栏工作台、移动端分段标签页
```

交互关键词：

```text
点击知识点即筛选、勾选错题生成练习、复习结果即时更新、状态标签可编辑、列表和详情联动
```

## 可直接用于设计工具的页面描述词

```text
设计一个独立的 AI 错题复习工作台页面，面向学生、家长和教培老师。页面首屏直接展示今日待复习、逾期复习、本周新增、已掌握等学习状态，并突出“拍试卷入库”和“生成今日练习”两个主要动作。页面中部展示薄弱知识点 Top 5、错因分布和重复出错提醒，帮助用户从错题中看见学习问题。主体区域包含可筛选的错题列表、错题详情面板和练习生成配置。错题列表支持按学科、年级、知识点、错因、题型、难度、掌握状态搜索和筛选；详情面板展示原题、学生答案、正确答案、AI 解析、知识点标签、错因标签、复习记录，并提供“做对了”“又错了”“已掌握”“生成同类题”等操作；练习生成支持选择今日待复习、选中错题或指定知识点，配置题量、难度、题型，并输出在线预览和 PDF 下载。整体风格清爽可信、低焦虑、信息层次清楚，像学习工具而不是后台系统；适合家长快速理解，也适合学生每天复习使用。
```

## 页面文案备选

- 页面标题：错题复习工作台
- 副标题：把每一道错题变成下一次练习的入口
- 今日任务：今天优先复习这些题
- 薄弱点：最近反复出错的知识点
- 错因分析：不是只看错了什么，也看为什么错
- 练习生成：按当前薄弱点生成一份巩固练习
- 空状态：完成一次错题解析后，这里会自动生成复习计划
