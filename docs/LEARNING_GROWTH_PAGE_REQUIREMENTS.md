# 错题与学习成长页面重设计需求说明

## 1. 需求背景

当前系统已经具备错题识别、错题入库、错题列表、错题详情、复习记录和练习生成能力。下一步页面重设计不改变现有核心功能，而是把这些能力重新组织成一条更清晰的学习闭环：

```text
错题识别 -> AI 批改与解析 -> 错题沉淀 -> 针对性练习生成 -> 复习反馈 -> 学习成长曲线
```

页面目标不是新增复杂功能，而是让用户更直观地理解：

- 最近错在哪里？
- 为什么会错？
- 今天应该练什么？
- 练习后有没有提高？
- 一段时间内学习状态是否在变好？

## 2. 设计原则

### 2.1 现有核心功能保持不变

以下能力在页面重设计中必须保留：

- 上传已批改试卷或题目图片，识别错题。
- AI 输出题干、学生答案、正确答案、解析、知识点、错因、题型、难度。
- 错题自动入库。
- 错题列表筛选与搜索。
- 错题详情查看与标签编辑。
- 记录复习结果：做对了、又错了、已掌握。
- 基于错题生成巩固练习卷。
- 支持练习卷预览、答案、解析、PDF 下载或打印。

### 2.2 页面表达重新设计

重设计重点放在信息架构和用户心智上：

- 从“工具集合”调整为“学习成长工作台”。
- 从“错题列表优先”调整为“今日任务与成长反馈优先”。
- 从“做完一次解析”调整为“持续复习、练习、提高”。
- 从“展示数据”调整为“告诉用户下一步该做什么”。

## 3. 目标用户

| 用户 | 使用场景 | 核心诉求 |
| --- | --- | --- |
| 学生 | 每天复习错题、生成练习、查看进步 | 少整理，知道今天该练什么，看到自己在提高 |
| 家长 | 上传试卷、查看薄弱点、监督复习 | 不懂题也能看懂孩子错因和成长趋势 |
| 老师 | 批量整理错题、按知识点出练习 | 快速发现共性薄弱点，减少重复组卷工作 |

## 4. 核心业务闭环

### 4.1 错题

错题是整个页面的数据基础。

用户通过拍照、上传或解析结果入库生成错题记录。每道错题需要展示以下信息：

- 原题题干
- 学生答案
- 正确答案
- AI 解析
- 知识点
- 错因
- 题型
- 难度
- 掌握状态
- 复习次数
- 下次复习时间

页面上应弱化技术字段，强化学习语义，例如“函数概念反复出错”“审题错误”“今天需要复习”。

### 4.2 批改

批改不只是判断对错，而是帮助用户理解错因。

批改模块需要表达：

- 哪几题错了。
- 学生答案和正确答案差异在哪里。
- 错误原因是什么。
- 关联哪些知识点。
- 是否建议加入复习计划。

批改结果需要能进入错题库，并成为后续练习生成和成长曲线的数据来源。

### 4.3 练习生成

练习生成是错题转化为提高动作的关键入口。

练习生成应支持现有模式：

- 根据今日待复习错题生成练习。
- 根据用户勾选的错题生成练习。
- 根据薄弱知识点生成专项练习。
- 生成同类变式题。
- 生成易错辨析题。

配置项保持：

- 题量：5 / 10 / 15 / 20
- 难度：简单 / 中等 / 困难 / 混合
- 题型：同类变式题 / 知识点专项题 / 易错辨析题
- 输出：在线预览 / 答案 / 解析 / PDF 下载或打印

页面表达上应突出“为什么生成这套练习”，例如：

```text
根据最近 7 天的函数错题和 3 道待复习题，为你生成 10 道同类变式题。
```

### 4.4 提高

“提高”不是单独按钮，而是错题复习和练习生成后的结果反馈。

页面需要通过以下指标表达提高：

- 待复习错题减少。
- 已掌握错题增加。
- 同一知识点重复错误次数下降。
- 练习正确率提升。
- 错因分布改善，例如“计算错误减少”。
- 按周或按月展示学习趋势。

提高模块应避免夸张承诺，不说“成绩必然提升”，而是表达为“掌握状态改善”“薄弱点减少”“练习正确率提高”。

### 4.5 学习成长曲线

学习成长曲线用于把零散错题转成长期反馈。

建议展示维度：

- 每日 / 每周新增错题数量
- 每日 / 每周完成复习数量
- 已掌握错题数量变化
- 待复习错题数量变化
- 练习正确率趋势
- 薄弱知识点数量变化
- 高频错因变化

曲线不需要复杂，MVP 页面可先展示 2 到 3 条关键趋势：

- 掌握度趋势
- 待复习压力趋势
- 练习正确率趋势

## 5. 页面信息架构建议

### 5.1 首屏：学习状态总览

首屏回答“今天应该做什么”。

建议模块：

- 当前学生 / 年级 / 学科
- 今日待复习
- 逾期未复习
- 本周新增错题
- 已掌握错题
- 今日推荐动作：生成今日练习、拍试卷批改

关键文案：

```text
今天优先处理这些错题
```

```text
根据最近错题，为你准备一组巩固练习
```

### 5.2 第二屏：错题与错因诊断

回答“最近主要错在哪里”。

建议模块：

- 薄弱知识点 Top 5
- 错因分布
- 重复出错提醒
- 题型 / 难度分布

交互：

- 点击知识点筛选错题。
- 点击错因查看相关题目。
- 点击“生成专项练习”进入练习配置。

### 5.3 第三屏：错题处理工作区

回答“具体是哪几题，怎么处理”。

建议模块：

- 错题列表
- 错题详情
- 标签编辑
- 复习操作
- 批量操作

列表字段：

- 题干摘要
- 知识点
- 错因
- 状态
- 下次复习时间
- 最近复习结果

详情字段：

- 原题
- 学生答案
- 正确答案
- AI 解析
- 知识点标签
- 错因标签
- 复习记录
- 生成同类题入口

### 5.4 第四屏：练习生成

回答“下一步练什么”。

建议模块：

- 练习来源：今日待复习 / 勾选错题 / 指定知识点 / 重复错误
- 练习目标：巩固基础 / 查漏补缺 / 考前复习 / 易错辨析
- 题量、难度、题型配置
- 生成结果预览
- PDF 下载 / 打印

### 5.5 第五屏：学习成长曲线

回答“这段时间有没有变好”。

建议模块：

- 掌握度趋势
- 练习正确率趋势
- 待复习压力趋势
- 错因改善趋势
- 知识点掌握变化

页面文案建议：

```text
过去 7 天，已掌握错题增加 12 道，待复习错题减少 8 道。
```

```text
函数相关错题仍然较多，建议继续生成专项练习。
```

## 6. 页面功能清单

| 模块 | 功能 | 是否保留现有能力 | 设计重点 |
| --- | --- | --- | --- |
| 错题识别 | 上传试卷图片并识别错题 | 是 | 入口更明确，结果更像批改报告 |
| AI 批改 | 展示答案、解析、错因、知识点 | 是 | 从“解析文本”升级为“诊断卡片” |
| 错题库 | 筛选、搜索、查看详情 | 是 | 信息更可扫描，减少后台感 |
| 复习记录 | 做对了、又错了、已掌握 | 是 | 操作更轻，反馈更及时 |
| 练习生成 | 按错题或知识点生成练习 | 是 | 突出生成依据和练习目标 |
| PDF 输出 | 练习卷下载或打印 | 是 | 保留为结果操作 |
| 成长曲线 | 展示学习趋势 | 可基于现有数据增强 | 强化提高反馈 |

## 7. 状态与指标

### 7.1 错题状态

- 待复习
- 复习中
- 已掌握
- 已归档

### 7.2 复习结果

- 做对了
- 又错了
- 已掌握

### 7.3 成长指标

- 总错题数
- 今日待复习数
- 逾期复习数
- 已掌握数
- 本周新增错题数
- 本周完成复习数
- 练习正确率
- 高频知识点
- 高频错因

## 8. 视觉与体验关键词

### 8.1 整体风格

```text
清爽可信、学习工作台、低焦虑、行动导向、成长反馈、家长可读、学生可用
```

### 8.2 信息表达

```text
今日任务优先、薄弱点可视化、错因诊断卡片、练习生成闭环、学习趋势可追踪
```

### 8.3 布局建议

桌面端：

```text
顶部学习状态概览 + 中部薄弱点诊断 + 下方错题列表/详情/练习生成三栏工作区 + 成长曲线区域
```

移动端：

```text
分段标签页：今日任务 / 错题 / 练习 / 成长；优先保证上传、复习、生成练习三个动作顺畅
```

### 8.4 避免方向

```text
不要做成纯后台表格，不要做成营销落地页，不要过度科技风，不要堆满图表，不要使用夸张提分承诺
```

## 9. 可直接用于页面设计的描述词

```text
设计一个面向学生、家长和老师的 AI 学习成长工作台页面，现有核心功能保持不变，包括错题识别、AI 批改解析、错题入库、错题筛选、复习记录、练习生成和 PDF 输出。页面需要把“错题、批改、练习生成、能力提高、学习成长曲线”串成一个完整闭环。首屏展示今日待复习、逾期未复习、本周新增、已掌握、练习正确率等学习状态，并提供“拍试卷批改”和“生成今日练习”两个主要动作。中部展示薄弱知识点 Top 5、错因分布、重复出错提醒，点击知识点或错因可以筛选相关错题并生成专项练习。主体区域包含错题列表、错题详情和练习生成配置，错题详情展示原题、学生答案、正确答案、AI 解析、知识点标签、错因标签、复习记录，并提供“做对了”“又错了”“已掌握”“生成同类题”等操作。练习生成支持今日待复习、勾选错题、指定知识点和重复错误四种来源，支持题量、难度、题型配置，并输出在线预览、答案解析、PDF 下载或打印。页面底部展示学习成长曲线，包括掌握度趋势、待复习压力趋势、练习正确率趋势和错因改善趋势。整体风格清爽可信、低焦虑、行动导向，像一个学习工作台而不是后台管理系统，让家长能看懂孩子哪里薄弱，让学生知道今天该练什么，并能看到持续提高的反馈。
```

## 10. 关键页面文案

- 页面标题：学习成长工作台
- 页面副标题：从错题批改到针对练习，看见每一次提高
- 今日任务：今天优先复习这些错题
- 错题诊断：最近主要错在哪里
- 批改反馈：看懂答案，也看懂为什么错
- 练习生成：根据薄弱点生成下一组练习
- 成长曲线：持续记录掌握度变化
- 空状态：完成一次试卷批改后，这里会生成错题、复习计划和成长趋势
